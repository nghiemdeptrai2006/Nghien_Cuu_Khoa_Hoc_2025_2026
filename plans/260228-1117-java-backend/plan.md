# Plan: NCKH Backend - Java Spring Boot & SQL Server
Created: 28/02/2026
Status: 🟡 In Progress

## Overview
Dự án nâng cấp hệ thống Quản lý Nghiên cứu Khoa học (NCKH) từ Web tĩnh sang Web Application hoàn chỉnh, sử dụng kiến trúc chuẩn Enterprise với Java Spring Boot và Microsoft SQL Server.

## Tech Stack
- **Frontend:** HTML/CSS/JS thuần, sử dụng Fetch API để gọi dữ liệu.
- **Backend:** Java 25, Spring Boot 3.x (Spring Web, Spring Data JPA, Spring Security).
- **Database:** Microsoft SQL Server (T-SQL).
- **Auth:** JWT (JSON Web Tokens).
- **IDE Thao tác:** IntelliJ IDEA & SQL Server Management Studio (SSMS).
