# Phase 01: Setup Environment (IntelliJ & SQL Server)
Status: ⬜ Pending
Dependencies: None

## Objective
Chuẩn bị sẵn sàng môi trường làm việc trên máy tính (IDE, JDK, Database Engine) để bắt đầu code Java Spring Boot.

## Requirements
### Functional
- [ ] IntelliJ IDEA được cài đặt và mở thành công.
- [ ] SQL Server Management Studio (SSMS) có thể kết nối vào `localhost` (Windows Authentication hoặc SQL Server Auth).

## Implementation Steps
1. [ ] Tải và cài đặt IntelliJ IDEA Community Edition (nếu chưa có).
2. [ ] Mở SSMS, tạo một Database mới tên là `NCKH_DB`.
3. [ ] (Tuỳ chọn) Tạo một Login user riêng trong SQL Server cho ứng dụng Spring Boot (vd: `nckh_user` / `password123`).

## Output
- Database `NCKH_DB` rỗng đã sẵn sàng trên SSMS.
- IntelliJ IDEA sẵn sàng tạo Project mới.
